Silverlight Sudoku Helper

Directions

  • The "big numbers" (black) show the values for each filled cell
  • The "small numbers" (gray) show the candidates for each empty cell
  • The arrow keys (Up, Down, Left, Right) move the yellow cursor to select a cell
  • The number keys (1-9) set the value of the selected cell
  • The Delete key clears the value of a filled cell
  • The C key toggles the display of candidates
  • The Escape key solves the puzzle automatically
  • Invalid moves are not allowed; conflicts are indicated by a red flash in the conflicting cell
  • Sudoku description and rules can be found in Wikipedia and Sudopedia

Notes

  • If keys stop working, click on the Sudoku board to restore keyboard focus to the application
  • The B key changes to a blank board suitable for copying a new puzzle from a book/web site/etc.
  • The W key changes to a sample board from Wikipedia (easy)
  • The S key changes to a sample board from Sudopedia (hard)
  • The F key changes to a nearly complete board (very easy!)
  • When copying a board, hold down the Shift key when typing the numbers to bold the given values
  • When guessing a value, hold down the Ctrl key when typing the number to italicize it
  • A number's display type (normal<->bold<->italicized) can be changed in-place; change the number's value by deleting the old value before typing the new one
  • For more information, see the blog post about Silverlight Sudoku Helper on Delay's Blog
